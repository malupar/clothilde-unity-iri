# Analysis

In this folder we have compiled several Python scripts that help show how the simulator performs

## Python Profiler

This script takes as input a csv that has the elapsed time for each simulate + connection and the time taken for each simulate call.

In order to generate this csv press 'T' in the middle of the Unity simulation and make sure the checkbox called "Profiling" works.

## sim_cusick

Simulates cusick test given the physical parameters of the cloth.

## sim_view

Simulates the manipulation of the cloth.

Press 'E' during the Unity simulation to start the recording and another time to stop it.
